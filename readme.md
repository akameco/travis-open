# travis-open

> Open travis in the browser


## Install

```
$ go get github.com/akameco/travis-open
```

## CLI

```
$ travis-open --help

  Usage:
    travis-open
    travis-open -h | --help
    travis-oepn --version

  Example:
    $ travis-open
```

## Usage

```
$ travis-open
```
